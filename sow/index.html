Introducción

En esta práctica se analizan cuatro sitios web reconocidos en plataformas de premios internacionales como Awwwards, Webby Awards y CSS Design Awards. El objetivo es aplicar la metodología Atelier evaluando estructura, usabilidad, visuales y desempeño.

1. Obys Agency

Estructura

Navegación vertical con animaciones suaves y jerarquía clara.

Usabilidad

Interacción intuitiva; el exceso de movimiento puede distraer.

Visuales

Minimalismo, tipografía expresiva y microinteracciones fluidas.

Desempeño

Carga moderada por vídeo, optimizada para navegadores modernos.

2. Active Theory

Estructura

Scroll horizontal con secciones diferenciadas.

Usabilidad

Transiciones coherentes y control del flujo de navegación.

Visuales

Uso intensivo de WebGL para experiencias inmersivas.

Desempeño

Requiere dispositivos potentes para rendimiento óptimo.

3. The Webby Awards

Estructura

Diseño editorial con menú fijo y módulos bien jerarquizados.

Usabilidad

Alta legibilidad; algunas secciones son densas en contenido.

Visuales

Paleta sobria, tipografía moderna, excelente responsive.

Desempeño

Buen tiempo de carga y optimización de recursos.

4. CSS Design Awards

Estructura

Catálogo visual con filtros y categorías claras.

Usabilidad

Navegación sencilla; scroll largo puede cansar.

Visuales

Diversidad de estilos con imágenes protagonistas.

Desempeño

Equilibrio entre carga de imágenes y fluidez.
